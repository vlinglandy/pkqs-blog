c++容器

vector

动态更新数组内存，底层实现是数组，可以按索引顺序访问，如果是插入和删除则需要内存区的拷贝

int初始化方式

代码 解释
vector a; 初始化一个名为a装有int值的vector容器
vector b(a); 用a定义b(地址不一样，完全是两个vector)
vector a(100); 定义一个a，有100个数字0
vector a(100,12); 定义一个a，有100个数字6

string初始化方式

vector s; 初始化一个名为s装有string值的vector容器
vector s(s.begin(),s.end()); 用s1定义s(地址不一样，完全是两个vector)
vector s(s1); 同上
vector s(100,”null”)； 初始化100个null

妙用

struct point{int x,y;} vector p; p用来存放坐标
vector a[MAXN] 定义多维数组

练习

初始化数值方式1：先定义空间大小，后循环访问

初始化数值方式2：直接初始化，不定义空间，然后push_back();

汇总：

list

STL中的list是双向链表，在内存中地址不连续，通过指针来实现数据的访问，插入和删除的的时间复杂度是常数级

map

底层是平衡二叉搜索树，效率高

set

底层实现是二叉搜索树，在竞赛中很常见访问每个元素的时间复杂度是O(logn)

代码 解释
set s; 初始化一个set的集合
s.insert(item); 将item插入s中
s.erase(); 将item删除,成功则1反之0
s.find(k); 查找k返回一个迭代器
s.count(k); 查找k存在返回1，不存在则0
s.clear(); 清除s中的数据
s.empty(); 判断s是否为空
s.lower_bound(k); 返回不小于k的迭代器
s.upper_bound(k); 返回不大于k的迭代器

dequeue

queue

队列，先进先出，后进后出

代码 解释
queue q; 初始化一个q的队列
q.front(); 返回队首元素
q.back(); 返回队尾元素
q.pop(); 队首元素出队
q.push(num); 把num放入队尾部
q.size(); 返回队列的大小
q.empty(); 判断队列是否为空

priority_queue优先队列

底层实现是二叉堆，队里有自己的排序方式，每次入队都会将优先级最高的放在队首

上述结果是9,8,7—-0，说明每次push,优先队列都会把最大的数字放到前面

less会让队列从队首到队尾是高到低排列，也就是数字大的优先级越高（默认）

greater会让队列从队首到队尾是低到高排列，也就是数字小的优先级越高

stack

先进后出的数据结构

代码 解释
stack sta; 初始化一个sta的栈
sta.push(num); 把数字num放入sta中
sta.pop(); 队首元素出栈
sta.top(); 查看队首元素
sta.size(); 返回栈的长度
sta.empty(); 判断栈是否为空
